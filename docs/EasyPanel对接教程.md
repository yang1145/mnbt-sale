# EasyPanel 对接教程

本教程说明如何在 MNBT-SALE 后台对接 **EasyPanel（康乐面板）**，实现自动开通、暂停、重置密码等操作。

系统通过插件 `plugins/host/easypanel` 调用 EasyPanel 的 WHM API（`/api/`）。

## 前置条件

1. 已成功安装并登录 MNBT-SALE 后台（`/admin`）
2. 已部署可用的 **EasyPanel / 康乐面板**，并能正常登录管理端
3. 本机 PHP 可访问面板 API（建议开启 `allow_url_fopen`，或确保网络可达）

## 一、在 EasyPanel 中准备对接信息

登录 EasyPanel / 康乐后台，准备：

| 信息 | 说明 | 填到本系统的字段 |
|------|------|------------------|
| 面板地址 | 域名或 IP，**不要**带 `http://` / `https://` | 主机 |
| API 安全码 | 面板 API 密钥（用于签名校验） | 安全码 |
| 端口 | 面板访问端口 | 端口 |
| HTTPS | 若面板使用 HTTPS 访问则开启 | 是否开启 SSL |
| 产品列表 ID | 康乐后台「产品列表」中对应产品的 ID | 产品配置里的「产品列表ID」 |

> 账号、密码字段在 EasyPanel 对接中一般可留空（本插件主要用 **安全码** 做 API 签名）。

### API 调用说明

系统实际请求地址形如：

```text
http(s)://主机:端口/api/?c=whm&a=add_vh&...
```

签名规则（插件内实现）：

```text
s = md5( 操作名 + 安全码 + 随机数 r )
```

例如开通：`md5("add_vh" + security + r)`。

## 二、在本系统添加服务器

1. 登录后台：`http://你的域名/admin`
2. 进入 **服务器** 管理页面
3. 点击 **添加服务器**，按下方填写

### 字段对照

| 本系统字段 | 填写内容 | 示例 |
|------------|----------|------|
| 名称 | 自定义名称，仅后台识别用 | `EasyPanel主节点` |
| 主机 | 面板地址（不含协议） | `ep.example.com` 或 `1.2.3.4` |
| IP | 服务器 IP（可选） | `1.2.3.4` |
| 安全码 | EasyPanel / 康乐 **API 安全码** | （从面板后台复制） |
| 端口 | 面板端口 | `3312` / `80` / `443` 等 |
| 是否开启 SSL | 面板为 HTTPS 时选 **开启** | 开启 / 关闭 |
| 账号 | 一般可留空 | — |
| 密码 | 一般可留空 | — |
| 服务器插件 | 选择 **`easypanel`** | `easypanel` |

保存后确认服务器列表中插件显示为 `easypanel`。

## 三、创建产品并填写产品列表 ID

1. 后台进入 **产品分类**，先建好分类
2. 进入 **产品**，添加产品并：
   - 选择对应 **服务器**（上一步添加的 EasyPanel 服务器）
   - 配置价格、周期、库存等
3. 插件为 `easypanel` 时，产品侧会出现：

| 配置项 | 说明 |
|--------|------|
| 产品列表ID | 康乐 / EasyPanel 后台设置的产品列表 ID |

该 ID 对应开通时 API 参数 `product_id`，必须与面板里真实存在的产品一致。

4. 保存产品，前台即可购买开通

## 四、验证对接是否成功

1. 使用前台测试账号登录，并确保余额充足（或产品开启首次免费）
2. 打开产品页，填写时长后点击 **立即开通**
3. 成功时应提示开通成功，并跳转到 `/user/order/{订单ID}`
4. 在 EasyPanel 后台确认已创建对应虚拟主机
5. 用户中心订单页可使用 **登录控制面板**、**重置密码**

## 五、插件能力一览

| 功能 | API 操作 | 说明 |
|------|----------|------|
| 开通 | `add_vh` | 创建虚拟主机 |
| 暂停 | `stop` | 停用主机 |
| 解除暂停 | `start` | 启用主机 |
| 终止 | `del` | 删除主机 |
| 重置密码 | `change_password` | 修改主机密码 |
| 一键登录 | 表单跳转 `/vhost/?c=session&a=login` | 用户控制面板 |

## 六、常见问题

### 1. 提示「该产品没有配置服务器」

产品未绑定服务器，请到产品编辑页选择服务器。

### 2. 开通失败，返回 JSON 错误

常见原因：

- **安全码** 错误
- **产品列表ID** 在面板中不存在或未启用
- 主机 / 端口 / SSL 与实际访问方式不一致
- 本系统服务器无法访问面板 API

可在服务器上测试连通性：

```bash
curl -I "http://主机:端口/api/"
```

### 3. 主机地址怎么填

只填域名或 IP，例如：

- 正确：`ep.example.com`
- 错误：`http://ep.example.com`
- 错误：`ep.example.com/vhost`

### 4. 能开通但无法一键登录

检查端口、SSL 是否与用户访问面板的方式一致；登录地址为：

```text
http(s)://主机:端口/vhost/?c=session&a=login
```

### 5. 产品列表 ID 在哪里看

在 EasyPanel / 康乐后台的 **产品列表 / 套餐管理** 中查看对应产品的数字 ID，填到本系统产品配置的「产品列表ID」。
