# 梦奈宝塔对接教程

本教程说明如何在 MNBT-SALE 后台对接 **梦奈宝塔** 主机系统，实现自动开通、暂停、续费等操作。

> 后台首页「介绍/教程」中也有简要说明，本文为完整版。

## 前置条件

1. 已成功安装并登录 MNBT-SALE 后台（`/admin`）
2. 已部署可用的 **梦奈宝塔** 系统，并能正常登录
3. 本机 PHP 已启用 `curl` 扩展（用于调用梦奈 API）

## 一、在梦奈宝塔中准备对接信息

登录梦奈宝塔后台，准备以下信息：

| 信息 | 说明 | 填到本系统的字段 |
|------|------|------------------|
| 访问地址 | 梦奈宝塔的域名或 IP，**不要**带 `http://` / `https://` | 主机 |
| 系统 API | 梦奈宝塔系统级 API 密钥 | 安全码 |
| 宝塔编号 | 梦奈宝塔里对应服务器的「宝塔编号」 | 账号 |
| 调用密钥 | 该服务器的「调用密钥」 | 密码 |
| 端口 | 非 80/443 时填写实际端口 | 端口 |
| HTTPS | 若地址为 HTTPS 访问则开启 | 是否开启 SSL |

## 二、在本系统添加服务器

1. 登录后台：`http://你的域名/admin`
2. 进入 **服务器** 管理页面
3. 点击 **添加服务器**，按下方说明填写

### 字段对照

| 本系统字段 | 填写内容 | 示例 |
|------------|----------|------|
| 名称 | 自定义名称，仅后台识别用 | `梦奈主节点` |
| 主机 | 梦奈宝塔地址（不含协议） | `panel.example.com` 或 `1.2.3.4` |
| IP | 服务器 IP（可选，便于备注） | `1.2.3.4` |
| 安全码 | 梦奈宝塔 **系统 API** | （从梦奈后台复制） |
| 端口 | 访问端口；80 可填 `80`，HTTPS 常见 `443` | `80` / `443` / `8888` |
| 是否开启 SSL | 地址为 HTTPS 时选 **开启** | 开启 / 关闭 |
| 账号 | 梦奈宝塔里的 **宝塔编号** | （服务器列表中查看） |
| 密码 | 该服务器的 **调用密钥** | （服务器列表中查看） |
| 服务器插件 | 选择 **`mnbt`** | `mnbt` |

### 插件扩展配置

保存后再次进入该服务器编辑页，若插件为 `mnbt`，会出现：

| 配置项 | 说明 | 建议值 |
|--------|------|--------|
| 梦奈宝塔数据库版本 | 对应梦奈数据库版本号 | `1.6x` → `16`，`1.7x` → `17`（默认 `17`） |

填写完成后保存。

### 请求地址说明

系统实际调用接口形如：

```text
http(s)://主机:端口/api/api.php
```

请确保：

- 主机、端口、SSL 配置正确
- 本系统服务器能访问该地址（防火墙、安全组放行）
- 安全码、宝塔编号、调用密钥无误

## 三、创建产品并关联服务器

1. 后台进入 **产品分类**，先建好分类（如「虚拟主机」）
2. 进入 **产品**，添加产品并：
   - 选择对应 **服务器**（上一步添加的梦奈服务器）
   - 配置价格、周期、库存等
3. 若插件为 `mnbt`，产品侧通常还需配置：

| 配置项 | 说明 | 单位 |
|--------|------|------|
| 产品类型 | `虚拟主机` 或 `CDN` | — |
| 空间大小 | 主机空间配额 | M |
| 数据库大小 | 数据库配额 | M |
| 月流量大小 | 月流量 | G |
| 绑定域名数 | 可绑定域名数量 | 个 |

4. 保存产品，前台即可购买开通

## 四、验证对接是否成功

1. 使用前台测试账号登录，并确保余额充足（或产品开启首次免费）
2. 打开产品页，填写时长后点击 **立即开通**
3. 成功时应提示开通成功，并跳转到 `/user/order/{订单ID}`
4. 登录梦奈宝塔，确认已自动创建对应主机账号

若失败，页面会返回可读错误信息（如「创建失败接口无响应」），请按下方排查。

## 五、常见问题

### 1. 提示「该产品没有配置服务器」

产品未绑定服务器，请到后台产品编辑页选择服务器。

### 2. 提示「产品关联的服务器不存在」

产品绑定的服务器已被删除，请重新选择有效服务器。

### 3. 提示「服务器插件文件不存在 / 未实现开通接口」

服务器插件应选择 `mnbt`，并确认目录存在：

```text
plugins/host/mnbt/mnbt.php
```

### 4. 提示「创建失败接口无响应或返回异常」

常见原因：

- 主机地址填了 `http://` 或多余路径
- 端口 / SSL 与实际访问方式不一致
- 安全码、宝塔编号、调用密钥填错
- 本机无法访问梦奈 API（网络、防火墙）
- 梦奈宝塔服务未启动或 API 未开启

可在服务器上用 `curl` 测试：

```bash
curl -I "http://主机:端口/api/api.php"
```

### 5. 能连通但开通失败

- 检查产品配置（空间、流量、域名数等）是否符合梦奈侧限制
- 确认梦奈宝塔数据库版本号是否填对
- 查看梦奈宝塔侧日志/错误提示

## 六、字段速查（与后台首页教程一致）

- **主机** → 梦奈宝塔的地址  
- **安全码** → 梦奈宝塔系统 API  
- **账号** → 宝塔编号（梦奈宝塔里该服务器的宝塔编号）  
- **密码** → 调用密钥（梦奈宝塔里该服务器的调用密钥）  
- **端口** → 非 80 时填写实际端口  
- **SSL** → 地址为 HTTPS 时开启
